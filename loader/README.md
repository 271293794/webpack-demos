
# webpack loader 加载器



#### 使用方法

```
npm install 
```

初始化后将 `template-test-loader` 文件放到 `node_modules`文件中。`template-test-loader` 文件中存放的就是我们要写的`loader`，只不过这里没有做任何处理，只是把文件内容返回了。

```
npm start
```
